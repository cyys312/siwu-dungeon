# 《四武地牢》游戏设计文档

2D 像素风地牢探险 Roguelite（对标《元气骑士》+《Hades》）。单文件 HTML5 (Canvas)，无外部依赖，浏览器直接可玩。

## 当前版本 v1.6（发布准备中）

> 本节为最新状态速览；下方 v0.5 及更早章节保留为历史设计记录，部分数字已被后续版本覆盖。

- **4 武器 × 3 流派 = 12 流派，共 200 个天赋**——全部为机制型（联动/触发/转化/条件增强），非纯数值堆叠。
- **觉醒双段**：单武器投入 ≥8 点解锁流派觉醒，≥14 点解锁**究极觉醒**（不动明王 / 剑神 / 元素支配 / 百步穿杨）。
- **暴击体系**：铸造词条 + 精准天赋 + 蓄力箭 + 背刺（`o.critBonus` 通用暴击通道）；通用易伤 `e.markT`（受伤 ×1.2）。
- **10 种 Boss** 循环（每循环血量 +90%）；**9 种小怪** + **7 种精英词缀**。
- **房间**：小怪 / 宝箱 / 商店（含铸造台）/ 挑战房（全精英）/ **宝库房（精英守卫 + 双宝箱）** / 出生房祭坛。
- **元进度**：死亡累积「魂」，主菜单魂之天赋树永久强化，localStorage 存档。
- **主动技能**：4 个键石天赋改 Q 键释放（陨星 / 暴风雪 / 落雷 / 奇点），各自独立冷却。
- 代码约 2600 行；所有天赋接入真实战斗钩子并经 DBG 确定性测试 + 多智能体对抗式审查。

## v0.5 更新要点

- **每把武器新增第三流派**（8→12 流派）：大锤·雷霆流(连锁闪电)、长剑·疾风流(突进斩)、法师·奥术流(穿透分裂)、游侠·陷阱流(翻滚埋雷)，天赋 53→69
- **修复掩体卡怪**：小怪不再生成在墙/掩体内；加"卡墙自救"把被击退嵌进墙的怪挤出来；贴脸攻击不再被墙角挡
- **房间更大**（18×12 → 26×18），地图扩到 122×68
- **小怪强化**：基础血量/移速/接触伤害上调，成长曲线更陡，每房怪数增加
- **攻击特效大幅增强**：叠加发光刀光、命中屏幕白闪、方向性火花、大锤命中冲击波环
- **新机制**：闪电链、地雷（翻滚埋/自动布）、奥术奇点（吸引+持续灼伤）、旋刃气旋、落雷

## 核心循环

选择武器 → 探索地牢（打怪房/宝箱房/商店房）→ 击杀获得经验 → 升级抽天赋卡（3选1）→
层末 Boss 战 → **击杀 Boss 必得 2 次升级** → 下一层 → **无限层数**，越深越强。

## v0.4 更新要点

- **无限关卡**：3 个 Boss 循环出现，每循环一轮 Boss 血量 +90%、更狂暴（"强化·"前缀）
- **怪物成长**：同名怪越深血越厚、接触伤害越高、移速略快；经验/金币奖励同步上涨
- **大而统一的房间**（18×12），地图扩到 90×50
- **掩体系统**：石柱障碍物可卡走位挡弹幕；**炸药桶**挨打即爆、连锁殉爆、敌我皆伤
- **Boss 强化招式**：史莱姆王落地放环形弹；骸骨将军狂暴二连冲斩+双重弹环；巫妖新增诅咒法阵
- **攻击特效**：扇形刀光、方向性命中火花、大伤害数字放大
- **天赋扩充至 53 个**（每流派上限大幅提高 + 11 个新祝福），天赋卡不再显示流派

## 操作

| 按键 | 功能 |
|---|---|
| WASD | 移动 |
| 左键 | 攻击——**自动瞄准最近的敌人**（弓：按住蓄力，松开射出；无敌人时朝移动方向） |
| 右键 | 流派技能：大锤=举盾格挡（需天赋解锁）；游侠=切换 弓↔匕首 |
| 空格 | 翻滚（短冲刺 + 无敌帧） |
| E | 交互（开宝箱 / 购买 / 下楼） |
| 1/2/3 或点击 | 选择天赋卡 |
| R | 死亡后重开 |

## 四武器 × 双流派

**开局选一把武器**；升级时天赋卡从该武器的两个流派池 + 通用池中随机抽 3 张，
玩家靠"刷到什么、选什么"塑造流派（可纯走一系，也可混合）。

### 1. 大锤（近战重击，慢速高伤大范围+击退）

**重装流**（防御反伤）：
- `J1` 举盾格挡（核心）：解锁右键举盾，正面吸收伤害并 100% 反弹，格挡时移速大减
- `J2` 荆棘壁垒 ×3：反弹伤害每级 +75%
- `J3` 岩肤 ×2：每级 +1 层护甲（每场战斗开始时获得，吸收 1 次伤害）
- `J4` 屹立 ×2：格挡时移速惩罚减半 / 格挡成功回血
- `J5` 完美格挡（键石）：举盾瞬间 0.25s 内格挡 → 免伤、反弹 250%、震晕周围敌人

**战士流**（攻击套盾）：
- `W1` 战意护盾（核心）：挥锤每命中 1 名敌人获得 1 层护盾（上限 2），护盾抵挡 1 次伤害
- `W2` 势大力沉 ×3：锤伤害每级 +1
- `W3` 巨人挥击 ×2：攻击范围每级 +20%
- `W4` 狂热 ×3：攻速每级 +15%
- `W5` 地震波（键石）：挥锤向前方释放地裂冲击波（获得远程能力）

### 2. 长剑（近战快攻）

**吸血流**：
- `B1` 血刃（核心）：每命中 5 次回复 1 点生命
- `B2` 剑舞 ×3：攻速每级 +15%
- `B3` 三段连击：连击第 3 斩伤害 ×2
- `B4` 血怒：生命低于 1/3 时攻速 +40%、吸血所需命中减半
- `B5` 剑气纵横（键石）：每次挥剑放出一道穿透剑气，斩碎远处的敌人

**御剑流**（飞剑远程，升级=剑更多更快）：
- `F1` 御剑术（核心）：召唤 1 把飞剑环绕，自动飞向附近敌人
- `F2` 剑冢 ×3：每级 +1 把飞剑
- `F3` 疾风剑心 ×2：飞剑速度与出击频率每级 +30%
- `F4` 锋锐 ×2：飞剑伤害每级 +1
- `F5` 万剑诀（键石）：飞剑命中后弹射至另一名敌人

### 3. 法师（远程法术，耗法力，基础攻击为奥术弹）

**火法**：
- `P1` 火球术（核心·主攻）：主攻击变为爆炸火球（范围伤害）
- `P2` 环火术 ×2：每级 +1 颗环绕自身的火球，触敌爆炸后自动再生
- `P3` 爆轰 ×2：爆炸范围每级 +30%
- `P4` 燃焰 ×2：火焰点燃敌人，持续灼烧
- `P5` 陨星术（键石）：每 8 秒在光标处坠落陨石（大范围高伤）

**冰法**：
- `I1` 冰川蔓延（核心·主攻）：主攻击变为冰川——冰锥沿地面向前蔓延，穿透并减速敌人
- `I2` 霜寒刺骨 ×2：对被减速敌人的所有伤害每级 +1
- `I3` 冰封领域 ×2：冰川蔓延变宽（每级两侧各 +1 列）
- `I4` 冻牢：被冰击中 3 次的敌人冻结 2 秒
- `I5` 暴风雪（键石）：每 7 秒以自身为中心刮起暴风雪（持续伤害+减速）

> 规则：`P1` 与 `I1` 同为"主攻改造"，选了其中一个后另一个不再出现在卡池（其余天赋可混搭成"冰火双修"）。

### 4. 游侠（自带双武器：弓 + 匕首，右键切换）

**弓流**（穿透）：
- `A1` 贯穿箭（核心）：满蓄力箭穿透所有敌人
- `A2` 多重射击 ×2：每级 +1 支侧向箭
- `A3` 迅捷蓄力 ×2：蓄力速度每级 +35%
- `A4` 锐利 ×2：箭伤害每级 +1
- `A5` 箭雨（键石）：满蓄力命中处天降箭雨

**匕首流**（隐身）：
- `D1` 影遁（核心）：翻滚后隐身 1.5 秒，敌人失去目标
- `D2` 背刺：隐身中的攻击伤害 ×3
- `D3` 轻功 ×2：移速每级 +12%
- `D4` 淬毒 ×2：匕首命中使敌人中毒（持续伤害）
- `D5` 影舞（键石）：背刺击杀立即重置翻滚冷却并再次隐身

### 通用天赋（低权重混入所有卡池）

- `G1` 生命之泉 ×3：最大生命 +1 并回满该格
- `G2` 疾行 ×2：移速 +10%
- `G3` 聚宝 ×2：拾取范围扩大，敌人掉落更多金币

## 房间系统

每层 10 个房间，**串字型链式布局**——房间沿蛇形路径一间连一间（出生 → …… → Boss），
路线清晰不迷路：

1. **小怪房**：进入后封门，杀光解锁；掉经验球（升天赋）+金币；清房奖励金币
2. **宝箱房**：无怪，按 E 开箱获得随机惊喜 buff（回满血 / 金币 / 免费天赋卡 / 临时双倍攻击 / 护盾）
3. **商店房**：3 个货台，按 E 购买——回复生命、最大生命+1、**武器强化+1伤害**、随机天赋
4. **Boss 房**（每层最深处）：击杀后必得 2 次升级 + 出现下行楼梯

## 数值基线（原型）

- 玩家：6 生命、移速 84、翻滚冷却 0.7s
- 经验：升级所需 = 6 + 4×(等级-1)；小怪 2~4 经验，Boss 额外必给 2 级
- 法师法力 100，回复 20/s；奥术弹 6、火球 15、冰川 12
- 金币：小怪 1~3，清房 +5，Boss +20；商店定价 12~35

## 美术 & 反馈

代码生成像素精灵（16px 格），调色板暗紫地牢 + 高饱和角色色；
屏幕震动、击中闪白、伤害数字、粒子、光照渐晕；WebAudio 合成音效。
