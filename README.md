# 四武地牢 · Pixel Dungeon

2D 像素风地牢探险 Roguelite。**单文件、零依赖**，浏览器直接可玩。

🎮 **在线试玩：<https://cyys312.github.io/siwu-dungeon/>**

## 怎么玩

双击 [index.html](index.html) 用浏览器打开即可（Chrome/Edge 均可）。

| 按键 | 功能 |
|---|---|
| WASD | 移动 |
| Shift | 疾跑 |
| 左键 | 攻击——自动瞄准最近的敌人（弓：按住蓄力、松开射出） |
| 右键 | 流派技：大锤=举盾格挡（需天赋） / 游侠=切换弓↔匕首 |
| 空格 | 翻滚（无敌帧） |
| Q | 释放主动技能（键石天赋：陨星 / 暴风雪 / 落雷 / 奇点，各自独立冷却） |
| E | 交互（开宝箱 / 购买 / 献祭祭坛 / 铸造武器 / 下楼梯） |
| 1/2/3 或点击 | 选天赋卡 |
| R | 死亡后重开 |

## 核心玩法

- **四把武器**开局任选：大锤 / 长剑 / 法师 / 游侠
- **每把武器三个流派**（共 12 流派、**200 个天赋**），升级抽卡（3 选 1）随机出天赋——刷到什么、构筑什么
  - 大锤：重装 / 战士 / 雷霆　长剑：吸血 / 御剑 / 疾风　法师：火 / 冰 / 奥术　游侠：弓 / 匕首 / 陷阱
  - 天赋**全是机制型**：联动、触发、转化、条件增强（如「盾山」化护甲为攻、「元素反应」燃烧+减速触发热震、「连锁死亡」处决残血邻怪）
  - **觉醒**：单武器投入 ≥8 点解锁流派觉醒，≥14 点解锁 **究极觉醒**（不动明王 / 剑神 / 元素支配 / 百步穿杨）
- **丰富房间**：小怪房 / 宝箱房 / 商店房（含**武器铸造台**打词条）/ **挑战房**（全精英）/ **宝库房**（精英守卫+双宝箱）/ 出生房**地牢祭坛**（献祭换强化）
- **精英词缀**（迅捷/易爆/分裂/再生/蛮力/荆棘/瞬闪）与掩体石柱、炸药桶（挨打即爆、连锁殉爆、敌我皆伤）
- 每层最深处有 **Boss**（共 **10 种**循环），击杀必得 **2 次升级**
- **元进度**：死亡累积「魂」，主菜单魂之天赋树永久强化（生命/金币/伤害/速度/幸运），存档本地保存
- **无限关卡**：越往下怪物越强、Boss 循环强化，挑战你能走多深

详细设计见 [DESIGN.md](DESIGN.md)。

## 开发调试

浏览器控制台有 `DBG` 调试接口：`DBG.start('mage')` 直接开局、`DBG.god()` 上帝模式、
`DBG.learn('F1')` 直灌天赋、`DBG.warpRoom('boss')` 传送、`DBG.killAll()` 清场。

## 技术

HTML5 Canvas，约 2600 行原生 JS：代码生成像素精灵、过程生成地牢、
WebAudio 合成音效、确定性可测试的游戏循环。无任何外部资源或依赖。
